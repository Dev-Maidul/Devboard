Question-1: What are the different ways to select an element in the DOM?

There are several types of way to select an element in DOM.It's depend on which types of functionality I want to add.For instance 1.getElementById() if need any specific one element in that situation we can use it. 2. getElementsByClassName() If we need multiple same type element it that case we can give a same class name and select them by this method. 3.getElementsByTagName() we can sellect all same tags by this selector. 4. querySelector() It will return me the first element of same class name of multiple elements. 5. querySelectorAll() Selects all elements that match a CSS selector (returns a NodeList). Without these there are also some selector like getAttribute(),parentNode, childNodes, firstChild, lastChild, nextSibling, previousSibling and so on.

Question-4: What is event bubbling in the DOM?

Event bubbling is one of the familiar technic in java script to add an event.There are mainly two phases in event bubbling.1. Captureing phase it work mainly from parent node to child node whearas it not find the target event. 2. Event bubbling event this phase work reversly from the target node/child to parent node. Without it, parent elemnts can listen for events triggerd on child elements. and we can stop the bubbling proces using stopPropagation() and stopImmediatePropagation();
